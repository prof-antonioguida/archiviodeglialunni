Configurazione · Passo 1 di 3
Crea il progetto Google Cloud
Per connettere l'app a Google Drive hai bisogno di credenziali API. Segui questi passaggi (una volta sola, richiede ~5 minuti).
1
Vai su console.cloud.google.com e accedi con il tuo account Google.
2
Crea un nuovo progetto: clicca sul selettore in alto → "Nuovo progetto" → dai un nome (es. fileshare-drive) → Crea.
3
Nel menu a sinistra vai su API e servizi → Libreria. Cerca "Google Drive API" e abilitala.
4
Vai su API e servizi → Credenziali → Crea credenziali → ID client OAuth 2.0. Tipo applicazione: Applicazione web.
5
In "URI di reindirizzamento autorizzati" aggiungi l'URL di questa pagina (es. ). Poi clicca Crea.
6
Vai su API e servizi → Schermata consenso OAuth. Tipo: Esterno. Compila nome app e email. In "Utenti di test" aggiungi la tua email Google. Salva.
Configurazione · Passo 2 di 3
Inserisci le credenziali
Copia e incolla le informazioni che trovi nella pagina delle credenziali Google Cloud.
Trovalo in Credenziali → il tuo client OAuth → "ID client"
Apri la cartella su drive.google.com → l'ID è nella URL: drive.google.com/drive/folders/QUESTO_ID. Se vuoi usare il root di Drive, lascia vuoto.
Configurazione · Passo 3 di 3
Accedi con Google
Autorizza l'app ad accedere a Google Drive. Verrai reindirizzato alla schermata di login Google.
L'app richiede solo le autorizzazioni per leggere e modificare i file di Google Drive. Le tue credenziali non vengono mai inviate a server di terze parti — tutto avviene direttamente tra il tuo browser e Google.
